# starter-golang
starter of golang project

## Setup project
```bash    
$ make setup
```

## Build
```bash    
$ make build
```

## Run
```bash    
$ make run
```

## Run Dev
```bash    
$ make watch
```


## Build to Docker
```bash    
$ make docker-run
```
